# Sorting & Tree Visualizer

A simple terminal application to visualize sorting algorithms and tree traversals step by step. 
You might enjoy this CLI application if you have an interest in algorithms or competitive programming. I was inspired by videos that compare different sorting algorithms side-by-side, and I thought it would be fun to create something similar right in the terminal. Moreover, since I am passionate about competitive programming, writing this program served as excellent practice for brushing up on my algorithms.


You can watch function of program in demo-video.

## Installation
### 1. Clone the repository

```bash
git clone https://github.com/your-username/sorting-visualizer.git
cd sorting-visualizer
```
### 2. Compile (I did it yet and you can skip this step)
#### Linux:
```bash
g++ main.cpp -o visualizer_for_linux
```

#### Windows:
```bash
g++ main.cpp -o visualizer_for_windows.exe
```
## Run
#### Linux:
```bash
./visualizer_for_linux
```
#### Windows:
```bash
visualizer_for_windows.exe
```

#### You can start program only when you in directory with executing file
### You can select directory by cd in terminal in windows in linux

## Features
- **Sorting:** Bubble, Selection, Insertion, Quick Sort
- **Trees:** BFS, DFS
- **Adjustable animation speed**
- **Random data generation**
