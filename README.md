# Sorting & Tree Visualizer

A simple terminal application to visualize sorting algorithms and tree traversals step by step.

## Installation
### 1. Clone the repository

```bash
git clone https://github.com/your-username/sorting-visualizer.git
cd sorting-visualizer
```
### 2. Compile
#### Linux:
```bash
g++ main.cpp -o visualizer
```

#### Windows (MinGW):
```bash
g++ main.cpp -o visualizer.exe
```
## Run
#### Linux:
```bash
./visualizer
```
#### Windows:
```bash
visualizer.exe
```
## Features
- **Sorting:** Bubble, Selection, Insertion, Quick Sort
- **Trees:** BFS, DFS
- **Adjustable animation speed**
- **Random data generation**
